# ADR-0006: 이미지 최소화 작업 - Alpine 및 멀티 스테이지 빌드

## Status
Accepted

## Context
Koyeb의 허용 이미지 크기(2000 MiB)를 초과하여 배포 실패가 발생했습니다. 첫 번째 최적화 노력에도 불구하고, 압축 이미지 크기가 아직 기준을 충족하지 못하고 있습니다.

## Decision
Docker 이미지를 더 작은 크기로 빌드할 수 있도록 Alpine 기반 및 멀티 스테이지 빌드를 적용합니다.
- **Alpine**: Python 이미지를 Alpine로 변경하여 기본 크기를 최소화.
- **멀티 스테이지 빌드**: 먼저 빌드 단계에서 종속성을 설치하고, 런타임 단계에서는 필요한 파일만 포함.

## Consequences
- Docker 이미지 크기를 2000 MiB 이하로 줄여 배포 가능성을 높임.
- 빌드 및 실행 환경을 명확히 분리하여 관리 및 배포 효율성 강화.
